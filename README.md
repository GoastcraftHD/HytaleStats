# BetterStats

**BetterStats** adds a UI panel that tracks and displays a wide range of player statistics, such as mobs killed, blocks mined, and total time played.

### Features
- In-game statistics UI panel
- **/stats** command to open the menu
- Mob kill count
- More to come soon

### Current Status
At the moment, the mod only tracks **mobs killed**.  
Additional statistics (blocks mined, playtime, and more) are actively planned and will be added soon.
